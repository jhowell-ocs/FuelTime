# SubAgent Documentation

This directory contains documentation generated by subagents during complex feature development and refactoring.

## Structure

- `[feature]_spec.md` - Research phase outputs containing specifications
- `[feature]_review.md` - Initial review phase outputs
- `[feature]_review_final.md` - Final review after refinement
- `[feature]_[date].md` - Timestamped versions

## Purpose

These documents provide context and traceability for multi-phase implementations using the orchestrator workflow pattern. Each document captures:

- Research findings and best practices
- Architecture decisions
- Implementation details
- Quality assurance reviews
- Refinement recommendations

## Usage

These files are automatically created by subagents and referenced throughout the development workflow. Do not manually edit these files unless updating documentation after completion.
